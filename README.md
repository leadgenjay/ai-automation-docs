# AI Automation Insiders - Workflow Documentation

Interactive documentation templates for n8n workflows.

## Workflows Included

- **Google Ads Optimization Agent** - Weekly AI-powered campaign analysis

## Deployment

Hosted on Vercel. Auto-deploys on push to main.

## Adding New Workflows

1. Duplicate an existing HTML file
2. Update the workflow data (title, stages, credentials, config fields)
3. Push to main
